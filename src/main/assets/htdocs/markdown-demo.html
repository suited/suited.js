Suited's markdown plugin is based on markdown-it and so can use other markdown-it plugins for special mark-up. Suited's Markdown plugin add special markdown syntax to allow you to easily add fif=gures and slides using markdown.

# Special markdown syntax for figures and slides

For figures surround with your markdown like so
~~: **your bold stuff** :~~

For slides surround your markdown like so
~~* **your bold stuff** *~~

NB that if the ~~: and :~~ is on one line it is an inline-figure.

NB for block level figures put ~~: on a line by itself and follow with :~~ on a line by itself.
