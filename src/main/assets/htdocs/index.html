Introducing

Suited

Turn any document into a stylish presentation

Suited is a JavaScript library which turns documents into presentations! No longer do you have to separate the detailed information from the presentation. The article becomes the presentation.

The article is a presentation and the presentation is the article!

Lecturers can use slides to teach the class and distribute the slides and the details at the same time in one document. Business presentations can now be detailed documents, which can be published and colleagues can see the slides at the click of a button!

Traditional presentations have slides with more details embedded as speaker notes. Suited takes the opposite approach - a full document with embedded presentation slides. The slides can either be existing figures, graphs or quotes in the document, or hidden slides only visible in presentation mode.

Modes

Suited has four built in modes:

Built-in modes


  • Document - Normal reading mode
  • Presentation - Slides for presenting the information
  • Walkthrough - Highlight sections on navigation
  • Lecture - Scroll to sections on navigation and allow zoom of section
Give it a go! Press 'SHIFT+S' now to cycle through the modes
Press 'Esc' to jump directly to document mode
Press 'SHIFT+1' to 'SHIFT+9' to jump directly mode 1 to 9

There is no need to start with the presentation in mind, any document can be transformed into a presentation by adding slides, or marking sections that should be displayed in slide mode.

Embedded (Invisible) Slides

Use 
data-figure

Slides can be embedded in the document so that it is only visible when in deck mode. This means you can summarize a particular paragraph by embedding a slide in the document. To do that you add a section right after the paragraph with the slide details. To hide it you add the data-slide. This puts the section into 'slide only' mode which hides it when viewing the full document.

The slide becomes visible when the document is placed in deck mode. There is two ways to do this.

  • press SHIFT+S (to make a captial S)
    • This cycles through all the modes
  • Change mode in URL to be mode=deck
